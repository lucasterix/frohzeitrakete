# App-Icon Beschreibung

## Spezifikation fuer die externe Erstellung

**Hintergrund:** Sattes Gruen (#4F8A5B), einfarbig, keine Verlaeufe.

**Motiv:** Eine weisse, schlichte Rakete im Flat-Design-Stil. Die Rakete zeigt diagonal nach rechts oben. Sie besteht aus klaren, geometrischen Formen — kein Schlagschatten, keine feinen Details. Der Stil ist modern und minimalistisch, vergleichbar mit SF Symbols oder Material Icons.

**Ecken:** Abgerundete Ecken gemaess Apple Human Interface Guidelines (das System wendet das Masking automatisch an, das gelieferte Icon sollte quadratisch ohne Eckenradius sein, 1024x1024 px).

**Kein Text im Icon.** Apple lehnt Icons mit eingebettetem Text ab.

**Dateiformate:**
- 1024x1024 px, PNG, sRGB, kein Alpha-Kanal
- Keine Ebenen, keine Transparenz

**Stilreferenz:** Sauber, professionell, auf weissem und dunklem Hintergrund gut erkennbar. Die weisse Rakete auf dem gruenen Grund soll Aufbruch und Zuverlaessigkeit vermitteln.
